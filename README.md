# Interval timer

Interval timer for following turbo trainer workouts 

![](screenshot.png)

Can upload a .yml or .csv file describing the workout e.g.

```csv 
2:30,Warm-up
10:00,Int 1 160-170BPM
3:00,Rest
10:00,Int 2 160-170BPM
5:00,Rest
3:00,Hard gear 1/2
2:00,Rest

3:00,Hard gear 2/2
3:00,Cooldown
```
